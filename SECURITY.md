# Security Policy

## Supported versions

This project is pre-1.0. Only the latest published version receives security fixes.
There is no LTS policy yet.

| Version | Supported |
|---|---|
| latest | Yes |
| older | No |

## Reporting a vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email kaneshir97006@gmail.com with:
- A description of the vulnerability
- Steps to reproduce
- The version affected
- Any suggested fix (optional)

We aim to acknowledge reports within 72 hours and will follow up with a timeline
for a fix. We will credit reporters in the release notes unless you prefer to
remain anonymous.

## Scope

This policy covers the generic engine in this repository.

Vulnerabilities in consumer app adapters (implementations that live in other repos)
are out of scope — please report those to the respective maintainers.
